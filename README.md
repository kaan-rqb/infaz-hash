# 🔐 INFXZ HASH TOOL

INFXZ Hash Tool, dosya ve metinlerin hash değerlerini üretmek, doğrulamak ve karşılaştırmak için geliştirilmiş **eğitim amaçlı siber güvenlik aracıdır**.

---

## ⚠️ Uyarı

Bu araç **sadece eğitim ve öğrenme amaçlıdır**.  
Herhangi bir yasa dışı kullanımdan geliştiriciler sorumlu değildir.

---

## 🚀 Özellikler

- 🧾 Metin Hashleme (MD5, SHA1, SHA224, SHA256, SHA384, SHA512)
- 📁 Dosya Hashleme (büyük dosya desteği)
- ✔️ Hash Doğrulama
- 🔁 Dosya Karşılaştırma
- 📂 Klasör Tarama ve toplu hash çıkarma
- 📊 TXT ve JSON rapor oluşturma
- 🕘 İşlem geçmişi sistemi
- 📜 Log kayıt sistemi
- ⚙️ Ayarlar menüsü
- 🎨 Renkli terminal arayüzü (Termcolor)
- 🧠 Basit ve hızlı kullanım
